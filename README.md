# Leaf Disease Detection Using Machine Learning

## Project Overview

This project builds a machine learning model to detect and classify diseases in plant leaves.

Plant diseases can reduce agricultural productivity. Early detection helps farmers take preventive actions and improve crop yield.

---

## Dataset

The dataset contains information about plant leaves and their disease conditions.

Typical features include:

* Leaf color
* Leaf texture
* Spots on leaves
* Disease category

Target variable:

* Type of leaf disease

---

## Project Workflow

### 1 Data Preprocessing

* Data cleaning
* Handling missing values
* Feature scaling

### 2 Exploratory Data Analysis

* Distribution of leaf diseases
* Feature relationships
* Visualization using graphs

### 3 Machine Learning Models

The following models were tested:

* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine

---

## Model Evaluation

Models were evaluated using:

* Accuracy
* Confusion Matrix
* Precision
* Recall
* F1 Score

Random Forest provided the best performance for disease classification.
